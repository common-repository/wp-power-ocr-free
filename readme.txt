== Description ==

Extract easily text form your documents and images directly into Wordpress!
A pro version with more options is available: [https://www.info-d-74.com/en/produit/wp-power-ocr-plugin-wordpress-2/](https://www.info-d-74.com/en/produit/wp-power-ocr-plugin-wordpress-2/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the WP Power OCR and follow instruction to get your a free licence code
